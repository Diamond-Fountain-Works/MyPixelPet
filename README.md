# MyPixelPet

An iOS SwiftUI pixel pet application with shared models, views, audio, and
motion services.

## Project

Open `mypixelpet.xcodeproj` in Xcode and run the `mypixelpet` scheme.

The source tree is organized under:

- `mypixelpet/Shared`: shared models, services, resources, and views
- `mypixelpet/iOS`: iOS app entry point
- `mypixelpetTests`: unit tests
- `mypixelpetUITests`: UI tests

The Pages document is retained as the private project specification.

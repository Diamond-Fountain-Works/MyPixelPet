## Canonical Summary

MyPixelPet is a SwiftUI virtual pet project focused on a macOS floating desktop companion. The app renders a transparent, draggable pet overlay with menu bar controls and shared pet state for future iOS, macOS, and Android versions.

## 中文摘要
